# 🧑🏻‍💻 使用方式

> Please Refer to [RoboTwin 2.0 Document (Usage)](https://robotwin-platform.github.io/doc/usage/index.html) for more details.

## 比赛任务
本次比赛设置`place_dual_shoes`, `place_empty_cup`, `put_bottles_dustbin`, `stack_blocks_three`以及`stack_bowls_three`共5个任务，在评测时使用**demo_randomized**的任务设置进行评测。

## 关于比赛提交在线测试
在通过[挑战杯官网](https://2025.tiaozhanbei.net)报名并通过校团委和发榜单位审核后，请在[地瓜机器人在线测评网站](https://developer.d-robotics.cc/tiaozhanbei-2025) 注册并提交报名表，随后将获得API_KEY和提交工具。通过该工具提交结果至在线测试，并获取得分和排名。

5项任务中的每项任务，将在5个未公开的环境（背景 & 桌面纹理）中分别测试10次，共50次，取平均分，之后5项任务的得分相加得到总分（总分为100分）。每个环境的桌面上均存在随机的、未公开的干扰物体。同时，环境中存在桌面高度扰动和光照扰动，这部分的设置参考挑战杯分支中 task_configs/demo_randomized.yml 中的设置。

在7月31日之前，在线测试平台将开放不计提交次数的简单评价，即空白桌面且无干扰物体的环境供大家调试。8月1日中午12时起，切换正式测试环境，在正式测试环境中，每个队伍最多可以提交5次。

调试版在线测试平台预计于7于11日24时前开放提交。

在RoboTwin提供的baseline之外，发榜单位提供的baseline链接为[SEM](https://github.com/HorizonRobotics/robo_orchard_lab/tree/master/projects/sem/robotwin).

## 数据采集

### 1. 任务运行与数据采集

```
bash collect_data.sh ${task_name} ${task_config} ${gpu_id}
# Example: bash collect_data.sh place_empty_cup demo_randomized 0
```
